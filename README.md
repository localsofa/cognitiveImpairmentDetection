# Fatigue Detection Using Speech
side project i'm working on that will (hopefully) be able to detect fatigue and (later on) delirium in patients. this is an expansion of my "Acoustic Stress Detection in Driver Monitoring Systems" research, just more clinical :)

## Current Features:
* features.py: extracts acoustic / statistic features & adds them to a dataframe
    * speech rate; articulation rate; pause amount; pause length; pitch

## Data:
* Self-recorded database

## Later:

* ARS & transcriptions for word repetition, filler words, etc
* identification of key features, decision threshold
